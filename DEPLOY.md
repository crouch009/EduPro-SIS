## 🚀 Deploy Your Own

This is a **static single-page application** - no backend needed!

### Option 1: GitHub Pages
1. Clone this repository
2. Upload to GitHub
3. Enable GitHub Pages in Settings

### Option 2: Netlify Drop
1. Go to https://app.netlify.com/drop
2. Drag the `dist` folder
3. Done!

### Option 3: Vercel
1. Import from GitHub
2. Deploy automatically

## 🛠️ Tech Stack

- **React 18** with TypeScript
- **Tailwind CSS** for styling
- **Recharts** for data visualization
- **Lucide Icons** for beautiful icons
- **Vite** for building

## 📦 Single File Deploy

The entire app is bundled into a **single `index.html` file** (~232 KB gzipped) that contains all HTML, CSS, and JavaScript. No external dependencies needed!

## 🎨 Customization

- School name (Arabic & English)
- School logo (upload any image)
- School tagline
- Primary & accent colors
- Contact information
- Academic year & currency
- Social media links
- And much more!
